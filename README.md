# 🚦 REAL-TIME TRAFFIC MONITORING AND INCIDENT ALERT SYSTEM USING COMPUTER VISION

## 📌 Overview

This project presents a real-time traffic monitoring and alert system using **YOLOv11** and **ByteTrack**.
It detects, tracks, and analyzes vehicles from video streams and generates alerts for events like **overspeeding**.

---

## ⚙️ Features

* 🚗 Real-time vehicle detection using YOLOv11
* 🔁 Multi-object tracking using ByteTrack
* 📊 Vehicle speed estimation
* 🚦 Traffic density & congestion analysis
* ⚠️ Overspeed alert detection
* 📩 Telegram alert notification system

---

## 🧠 Tech Stack

* Python
* OpenCV
* YOLOv11
* ByteTrack
* Telegram Bot API
